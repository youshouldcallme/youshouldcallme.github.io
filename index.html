Here is how the software flows.

Once the call is set up in the bot's queue, the bot finds an optimal time to set up a call for the next day. At a random time, the bot will text the user and ask if now is a good time for the call. If the user responds, the bot will call the user and connect the user to the other party. If there is no response, the bot will try again in 10 minutes.
